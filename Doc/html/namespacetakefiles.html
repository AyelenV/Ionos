Parámetros

  file_type,
  year_select,
  mon_select,
  doy_select 
)

Extraer de la base de datos LPIM los nombres de los archivos con la data solicitada por usuario, con path absoluto.

Parámetros
path0: str : path al directorio que contiene los archivos fuente de datos (LPIMdatase)
file_type: str : tipo de archivo que contiene la data solicitada
year_select:list[str] : rango de años de la data solicitada por usuario
mon_select: list[list[str]] : lista de meses en correspondencia con los años solicitados.
doy_select: list[str] : rango de dias solictados (solo valido si usuario pide un mes en particular)
Devuelve
procesar : list[str] : lista de nombres de los archivos disponibles (path absoluto + archivo), para rango de fechas solicitados
Nota
Tipos de archivo disponibles en LPIMdatabase: 'prm', 'loc', 'est'.
