Descripción detallada

Esta Clase construye mapeos de los parametros ionosfericos, respecto a las variables (x,y).

Los parametros son calculados con el modelo LPIM (La Plata Ionospheric Model) y los mapas realizados en escala de color.

Calcula estadistico de la data, sobre el plano (x,y) dividido en bins de tamanno ajustable.

Parámetros
binsize: [int, int] : dimension del bin, [dx, dy].
statis: str : estadistico para scipy.stats.binned_statistic_2d
Nota
En scipy.stats.binned_statistic_2d
* el orden de los argumentos es y, x, z
* bins = [nrobinsx,nrobinsy] o bins = [xedges,yedges]
Devuelve
X (N,J) : 2D np.array : variable ordenada grillada
Y (J,M) : 2D np.rrray : variable abscisa grillada
Z_binned (M,N): 2D np.array : Data promediada sobre meshgrid (X,Y)
